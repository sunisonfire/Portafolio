### 🍰 Delicias del Ayer *(Junio 2026)*

**Desarrollo Web con Inteligencia Artificial**

> *"Una pastelería que desbloquea tus recuerdos con tan solo un mordisco."*

Delicias del Ayer es una página web diseñada para una pastelería que busca presentar sus productos de forma atractiva sin mostrar precios, permitiendo que los clientes conozcan la marca, su historia y su catálogo antes de realizar una compra.

El proyecto incorpora un carrusel interactivo, catálogo de productos, carrito de compras y una libreta de reseñas personalizada que reemplaza los sistemas de comentarios genéricos, aportando mayor confianza y credibilidad a la experiencia del usuario. Además, el catálogo se encuentra conectado a Google Sheets, permitiendo actualizar la información de los productos de manera dinámica sin necesidad de modificar el código fuente.

Este proyecto fue desarrollado mediante técnicas de Ingeniería de Prompts, enfocándose en la creación y optimización de instrucciones para generar una solución funcional, visualmente atractiva y adaptable a distintos dispositivos.

**Tecnologías utilizadas**

* Inteligencia Artificial
* HTML5
* CSS3
* JavaScript
* Google Sheets

### 🐾 Happy Paws *(Mayo 2026)*

**Diseño Web con HTML y CSS**

> *"¡Cada patita merece ser feliz, encuentra aquí a tu próximo compañero de vida!"*

Happy Paws es una plataforma web orientada a promover la adopción responsable de mascotas, así como la participación de voluntarios y donantes en refugios de animales.

El proyecto fue desarrollado completamente con HTML y CSS, centrándose en la construcción de una interfaz amigable, atractiva y accesible. Aunque sus funcionalidades son sencillas, destaca por su diseño visual, organización de contenido y experiencia de usuario, demostrando habilidades en maquetación web y diseño responsive.

**Tecnologías utilizadas**

* HTML5
* CSS3
* Diseño Responsive

### 🚌 KidRoute *(Mayo 2026)*

**Gestión de Transporte Escolar**

> *"La seguridad y organización de las rutas escolares en un solo lugar."*

KidRoute es una aplicación web desarrollada para administrar rutas de transporte escolar de manera eficiente y organizada.

Permite registrar conductores, crear rutas, asignar estudiantes y gestionar información como horarios de salida, destinos y documentación de cada usuario. Además, integra una API de clima para mostrar información meteorológica en tiempo real, facilitando la planificación de recorridos.

Este proyecto tuvo como objetivo fortalecer conocimientos en JavaScript mediante el uso de manipulación del DOM, consumo de APIs, validación de formularios y ventanas modales, complementados con HTML y CSS para la construcción de la interfaz.

**Tecnologías utilizadas**

* JavaScript
* HTML5
* CSS3
* DOM
* API de Clima

### 🐼 PANDA *(Junio 2026)*

**Coach Financiero Inteligente**

> *"Tus problemas financieros han llegado a su fin con tu nuevo coach virtual."*

PANDA es una plataforma de educación y acompañamiento financiero diseñada para ayudar a las personas a mejorar sus hábitos económicos mediante herramientas interactivas y asistencia impulsada por Inteligencia Artificial.

La aplicación ofrece funcionalidades enfocadas en ahorro activo, educación financiera automatizada, proyección de crecimiento financiero mediante interés compuesto, seguimiento de metas y una bitácora emocional que ayuda a identificar patrones relacionados con compras impulsivas.

El proyecto combina tecnología, experiencia de usuario y educación financiera para brindar una solución práctica que facilite la toma de decisiones económicas más conscientes e informadas.

**Tecnologías utilizadas**

* Inteligencia Artificial
* HTML5
* CSS3
* JavaScript
* Diseño UX/UI
